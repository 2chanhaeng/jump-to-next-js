## VSCode

[Visual Studio Code](https://code.visualstudio.com/) 는 MS 에서 만든 텍스트 에디터이다.  
쉽게 말하면... 이쁜 메모장이라는 뜻이다.  
물론 그 활용 범위는 무궁무진하다.  
만약 본인이 사용하는 IDE가 있다면 그걸 사용해도 된다.  
IDE가 뭔지 모르면 VSC를 사용하자.

## 설치

[VSCode](https://code.visualstudio.com/) 사이트에 들어가서 다운로드 받으면 된다.  
맥 사용자라면 아까 nvm 을 설치할 때 사용했던 [Homebrew](https://brew.sh/) 를 사용해서 설치할 수도 있다.  
다음 명령어를 터미널에 입력하면 된다.

```bash
brew install --cask visual-studio-code
```
